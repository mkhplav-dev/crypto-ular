О проекте

CryptoUlarbek — один человек, одна идея

Веб-инструмент для анализа крипторынка, созданный с нуля одним человеком. Без команды, без инвесторов — только чистая страсть к криптовалютам.

Вступление

Этот проект создан одним человеком с нуля, без команды и инвесторов. Идея появилась из простой мысли: криптовалюта слишком сложна для обычных людей. Большинство сервисов созданы для трейдеров, а не для простых пользователей.

Этот сайт — попытка изменить это. Сделать криптовалюту ближе и понятнее для каждого человека в Кыргызстане.

      Справка

      Частые вопросы

      Откуда берутся цены?

      Используются публичные API Binance (основной источник), данные из ForkLog и локальных крипто-сайтов (netex.kg, xbt.kg). Данные обновляются автоматически каждые 60 секунд.

      Насколько точен прогноз?

      Линия прогноза построена на методах MA и линейной регрессии и не является инвестиционной рекомендацией.

      Где хранится избранное?

      Список избранных монет сохраняется локально в браузере (localStorage), без сервера.
